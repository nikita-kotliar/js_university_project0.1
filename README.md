# Курсова робота з JavaScript

## Проєкт: Робочий веб-застосунок з вправами для тренувань

Це веб-застосунок, який дозволяє переглядати різноманітні вправи, додавати їх у список улюблених, оцінювати та отримувати детальну інформацію про кожну вправу.

## Функціонал

### Перегляд вправ

Відображення списку всіх доступних вправ.

Кожна вправа містить зображення (GIF), назву, рейтинг, цільові м’язи, обладнання, популярність та спалені калорії.

### Модальне вікно з деталями вправи

Відкривається при кліку на вправу.

Можливість закриття через кнопку, клік на бекдроп або клавішу Esc.

Кнопка додавання/видалення з улюблених.

Кнопка для переходу до оцінювання вправи.

### Улюблені вправи

Додавання/видалення вправ з локального сховища.

Перегляд усіх улюблених вправ на окремій сторінці.

### Система оцінювання

Вибір кількості зірок.

Додавання відгуку та електронної пошти.

Відправка оцінки на сервер.

### Адаптивний дизайн

Працює на мобільних, планшетах та десктопах.
